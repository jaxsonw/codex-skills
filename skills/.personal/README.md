# Personal Skills

This folder is for Jaxson's own Codex skills.

Use it for workflows that should reflect personal defaults rather than generic OpenAI examples:

- preferred engineering workflow
- project setup conventions
- review and release standards
- writing tone and bilingual output preferences
- recurring deployment or ops procedures

Each skill folder should contain at least:

```text
skill-name/
└── SKILL.md
```

Recommended optional folders:

```text
agents/       # UI metadata
scripts/      # reusable commands
references/   # longer docs loaded only when needed
assets/       # images, templates, examples
```
